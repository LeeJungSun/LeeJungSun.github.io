Attribute

  • [name|=”value”]

    지정된 특성을 가진 요소를 지정된 문자열과 동일한 값 또는 해당 문자열로 시작하여 하이픈 (-)으로 선택

  • [name*=”value”]

    주어진 하위 문자열을 포함하는 값으로 지정된 속성을 갖는 요소를 선택

  • [name~=”value”]

    공백으로 구분 된 주어진 단어가 포함 된 값으로 지정된 속성을 갖는 요소를 선택

  • [name$=”value”]

    주어진 문자열로 정확히 끝나는 값으로 지정된 속성을 갖는 요소를 선택합니다. 비교는 대소 문자를 구분

  • [name=”value”]

    특정 값과 정확하게 일치하는 지정된 속성을 갖는 요소를 선택

  • [name!=”value”]

    지정된 속성을 갖지 않거나 지정된 속성을 가지나 특정 값이없는 요소를 선택

  • [name^=”value”]

    주어진 문자열로 정확히 시작하는 값으로 지정된 속성을 갖는 요소를 선택

  • [name]

    임의의 값으로 지정된 속성을 갖는 요소를 선택

  • [name=”value”][name2=”value2″]

    지정된 모든 속성 필터와 일치하는 요소를 선택
